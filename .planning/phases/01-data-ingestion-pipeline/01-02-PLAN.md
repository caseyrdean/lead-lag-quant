    **ingestion_massive/polygon_client.py:**
    - Class `PolygonClient` with `__init__(self, api_key: str, rate_limit_per_minute: int = 5)`:
      - Store api_key
      - Create `pyrate_limiter.Limiter` with Rate(rate_limit_per_minute, Duration.MINUTE)
      - Create `requests.Session()` with `HTTPAdapter(max_retries=Retry(total=5, backoff_factor=1, backoff_jitter=0.5, status_forcelist=[429, 500, 502, 503, 504], respect_retry_after_header=True))` mounted on "https://"
      - Use structlog logger via `get_logger("polygon_client")`

    - Private method `_get(self, url: str, params: dict | None = None) -> dict`:
      - Call `self.limiter.try_acquire("polygon_api")` BEFORE the request (pre-throttle)
      - Always inject `params["apiKey"] = self.api_key`
      - Call `self.session.get(url, params=params)`, call `response.raise_for_status()`
      - Return `response.json()`
      - Log request URL (without API key) at debug level

    - Private method `_paginate_v3(self, url: str, params: dict) -> tuple[list[dict], list[dict]]`:
      - Generic pagination for v3 endpoints
      - Accumulates results AND raw responses (each page's full JSON)
      - Follows `next_url` from response, setting `params = {}` for subsequent pages (cursor in URL, only apiKey added by _get)
      - Returns (all_results, all_raw_responses)

    - `get_aggs(self, ticker, from_date, to_date, timespan="day", multiplier=1) -> tuple[list[dict], list[dict]]`:
      - URL: `{BASE_URL}/v2/aggs/ticker/{ticker}/range/{multiplier}/{timespan}/{from_date}/{to_date}`
      - Params: `adjusted=false`, `sort=asc`, `limit=50000`
      - Pagination: follow `next_url`, set params={} for subsequent pages
      - Returns (all_results, all_raw_responses) -- raw responses is list of full JSON dicts from each page
      - CRITICAL: Always pass `adjusted=false` (INGEST-02)

    - `get_splits(self, ticker) -> tuple[list[dict], list[dict]]`:
      - URL: `{BASE_URL}/v3/reference/splits`, params: `ticker={ticker}`, `limit=1000`
      - Uses `_paginate_v3`
      - Returns (results, raw_responses)

    - `get_dividends(self, ticker) -> tuple[list[dict], list[dict]]`:
      - URL: `{BASE_URL}/v3/reference/dividends`, params: `ticker={ticker}`, `limit=1000`
      - Uses `_paginate_v3`
      - Returns (results, raw_responses)

    - `get_ticker_details(self, ticker) -> dict | None`:
      - URL: `{BASE_URL}/v3/reference/tickers/{ticker}`
      - Returns the `results` dict if `active` is True, else None
      - Catches `requests.HTTPError` and returns None (invalid ticker)

    BASE_URL = "https://api.polygon.io" as class constant.

    **ingestion_massive/ingestion.py:**

    Function `store_raw_response(conn, ticker: str, endpoint: str, request_params: dict, response_json: str) -> None`:
    - Serialize `request_params` to deterministic JSON: `json.dumps(request_params, sort_keys=True)`
    - Execute idempotent upsert:
      ```sql
      INSERT INTO raw_api_responses (ticker, endpoint, request_params, response_json, retrieved_at)
      VALUES (?, ?, ?, ?, datetime('now'))
      ON CONFLICT (ticker, endpoint, request_params)
      DO UPDATE SET response_json = excluded.response_json, retrieved_at = excluded.retrieved_at
      ```
    - Commit after insert

    Function `log_ingestion(conn, ticker: str, endpoint: str, date_from: str | None, date_to: str | None, status: str, records_fetched: int = 0, error_message: str | None = None) -> int`:
    - Insert into ingestion_log, return the row id
    - For status="completed", also set completed_at = datetime('now')

    Function `update_ingestion_log(conn, log_id: int, status: str, records_fetched: int = 0, error_message: str | None = None) -> None`:
    - Update the ingestion_log row with final status, records_fetched, completed_at, error_message

    Function `ingest_ticker(client: PolygonClient, conn: sqlite3.Connection, ticker: str, from_date: str, to_date: str, progress_callback=None) -> dict`:
    - Fetches aggs, splits, and dividends for a single ticker
    - For each endpoint:
      1. Log ingestion start (status="started")
      2. Call client method (get_aggs, get_splits, get_dividends)
      3. Store each raw response page via store_raw_response
      4. Update ingestion log (status="completed", records_fetched=len(results))
      5. On exception: update ingestion log (status="failed", error_message=str(e)), log error, continue to next endpoint
    - Returns dict: `{"ticker": ticker, "aggs": count, "splits": count, "dividends": count}`
    - If progress_callback provided, call it after each endpoint completes

    Function `ingest_pair(client: PolygonClient, conn: sqlite3.Connection, leader: str, follower: str, from_date: str, to_date: str, progress_callback=None) -> dict`:
    - Determines the unique set of tickers to fetch: {leader, follower, "SPY"} (INGEST-10: SPY always included)
    - Calls ingest_ticker for each ticker in the set
    - Returns dict with per-ticker results
    - Handles the case where leader or follower IS "SPY" (no duplicate fetch)
    - Uses structlog bound logger with pair context
