    - `get_connection(db_path: str | Path) -> sqlite3.Connection`: Create connection with `check_same_thread=False`, set `PRAGMA journal_mode=WAL`, `PRAGMA foreign_keys=ON`, set `row_factory = sqlite3.Row`

      - `ticker_pairs` (id, leader, follower, created_at, is_active, UNIQUE(leader, follower))

      - `ingestion_log` (id, ticker, endpoint, date_from, date_to, status, records_fetched, error_message, started_at, completed_at)
    - Use `CREATE TABLE IF NOT EXISTS` for idempotent schema creation
    - Call `conn.commit()` after executescript
    - Ensure `data/` directory is created (via `Path(db_path).parent.mkdir(parents=True, exist_ok=True)`) inside `get_connection`

Use ConsoleRenderer for dev, JSONRenderer for json_output=True. Set filtering bound logger at INFO level.
    - `get_logger(name: str)`: Return `structlog.get_logger().bind(module=name)`

    - `is_trading_day(date: str) -> bool`: Check if date is NYSE session
    - Cache the calendar instance at module level (it is expensive to create)
