## Decisions Made
- `httpx` added to `[dependency-groups].dev` — starlette's TestClient requires httpx directly; it was previously pulled in transitively by gradio and would have broken all API tests without this fix
- `reload=False` in uvicorn.run — background threads started in FastAPI lifespan are not compatible with uvicorn's reload mode (file watcher kills/restarts workers)
- Pre-existing test failures (test_engine_detector, test_signals_generator) confirmed out of scope — they reference modules already modified before this plan executed
