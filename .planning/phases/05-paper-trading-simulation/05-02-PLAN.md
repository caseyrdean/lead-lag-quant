**1. Create ui/signal_dashboard.py (UI-01):**

This module exports a single function `build_signal_dashboard_tab(conn, config)` that is called inside the gr.Blocks context in app.py. It builds a `gr.Tab("Signal Dashboard")` with all internal components and event wiring.

Structure:
```python
import gradio as gr
import pandas as pd
import sqlite3
from paper_trading.engine import auto_execute_signals, get_open_positions_display
from paper_trading.db import get_unprocessed_signals
from utils.logging import get_logger
```

Internal helper functions (defined inside build_signal_dashboard_tab or at module level, taking conn/config as closure or params):

- `get_active_signals_dataframe(conn) -> pd.DataFrame`: Query `SELECT rowid AS signal_id, ticker_a, ticker_b, signal_date, direction, sizing_tier, stability_score, correlation_strength, expected_target, invalidation_threshold, generated_at FROM signals WHERE signal_date >= date('now', '-7 days') ORDER BY generated_at DESC`. For each signal, add a boolean "Executed" column by checking if a paper_trades buy record exists for that signal_id. Return DataFrame with columns: Signal Date, Leader (ticker_a), Follower (ticker_b), Direction, Sizing Tier, Stability, Correlation, Target, Invalidation, Generated At, Executed. If no signals, return empty DataFrame with those columns.

- `refresh_signals_callback()`: Returns get_active_signals_dataframe(conn).

- `execute_signals_callback(auto_execute_enabled)`: If auto_execute_enabled is False, return "Auto-execute is OFF. Enable the toggle first." + unchanged table. Otherwise call auto_execute_signals(conn, config.polygon_api_key). Return status message (f"Executed {len(results)} signal(s)") + refreshed signals table.

Layout inside the gr.Tab("Signal Dashboard"):
```
gr.Markdown("## Signal Dashboard")
gr.Row:
    auto_execute_toggle = gr.Checkbox(label="Auto-Execute New Signals", value=False, info="When enabled, qualifying signals auto-open paper positions")

    value=refresh_signals_callback (called on load)
    headers=["Signal Date", "Leader", "Follower", "Direction", "Sizing Tier", "Stability", "Correlation", "Target", "Invalidation", "Generated At", "Executed"]
    interactive=False
    label="Active Signals (last 7 days)"
gr.Row:
    execute_btn = gr.Button("Execute New Signals", variant="primary")
    refresh_signals_btn = gr.Button("Refresh")
status_msg = gr.Textbox(label="Status", interactive=False)
```

Event wiring:
- execute_btn.click(fn=execute_signals_callback, inputs=[auto_execute_toggle], outputs=[status_msg, signals_table])
- refresh_signals_btn.click(fn=refresh_signals_callback, outputs=[signals_table])

**2. Create ui/paper_trading_panel.py (UI-04):**

This module exports `build_paper_trading_tab(conn, config)` -- called inside gr.Blocks context in app.py. Builds a `gr.Tab("Paper Trading")` with all components and event wiring.

```python
import gradio as gr
import pandas as pd
from datetime import datetime, timezone
from paper_trading.engine import (
    set_capital, open_or_add_position, close_position,
    get_portfolio_summary, get_open_positions_display,
    get_trade_history_display, compute_share_quantity,
)
from paper_trading.price_poller import poll_and_update_prices, fetch_snapshot_price
from paper_trading.db import check_exit_flags
from utils.logging import get_logger
```

Internal helper functions:

- `get_positions_dataframe() -> pd.DataFrame`: Call get_open_positions_display(conn). Convert to DataFrame with columns: Ticker, Shares, Avg Cost, Current Price, Unrealized P&L, Exit Flag. Format: Avg Cost and Current Price to 2 decimal places. Exit Flag as "EXIT" string or empty string. If no positions, return empty DataFrame with those columns.

- `get_history_dataframe() -> pd.DataFrame`: Call get_trade_history_display(conn). Convert to DataFrame with columns: Executed At, Ticker, Side, Shares, Price, Realized P&L, Notes. If no trades, return empty DataFrame.

- `get_summary_values() -> tuple`: Call get_portfolio_summary(conn). Return (cash_balance, total_pnl, win_rate). If no portfolio, return (0, 0, 0).

- `set_capital_callback(amount) -> tuple`: Call set_capital(conn, float(amount)). Return updated (status_msg, cash_balance, total_pnl, win_rate).

- `buy_callback(ticker, shares) -> tuple`: Validate ticker non-empty, shares > 0. Uppercase ticker. Fetch current price via fetch_snapshot_price(ticker, config.polygon_api_key). If price is None, return error status. Call open_or_add_position(conn, 1, ticker, int(shares), price, None, None, now_utc). Return updated (status_msg, positions_table, cash_balance, total_pnl, win_rate, history_table).

- `sell_callback(ticker, shares) -> tuple`: Validate ticker, shares. Uppercase ticker. Fetch current price. Call close_position(conn, 1, ticker, int(shares), price, now_utc). Return updated (status_msg, positions_table, cash_balance, total_pnl, win_rate, history_table). Catch ValueError and return error message.

- `refresh_prices_callback() -> pd.DataFrame`: Call poll_and_update_prices(conn, config.polygon_api_key). Return updated get_positions_dataframe(). This is the function connected to the gr.Timer tick.

Layout inside gr.Tab("Paper Trading"):
```
gr.Markdown("## Paper Trading")

# Capital setup row (TRADE-01)
gr.Row:
    starting_capital_input = gr.Number(label="Starting Capital ($)", value=100000, minimum=1000, step=1000)
    set_capital_btn = gr.Button("Set Capital", variant="secondary")

# Portfolio summary row
gr.Row:
    cash_display = gr.Number(label="Cash Balance", interactive=False, precision=2)
    pnl_display = gr.Number(label="Total P&L", interactive=False, precision=2)
    winrate_display = gr.Number(label="Win Rate (%)", interactive=False, precision=1)

# Open positions with 15-min auto-refresh (TRADE-04, TRADE-05)
price_timer = gr.Timer(value=900, active=True)
positions_table = gr.Dataframe(
    value=get_positions_dataframe,
    headers=["Ticker", "Shares", "Avg Cost", "Current Price", "Unrealized P&L", "Exit Flag"],
    label="Open Positions (prices refresh every 15 min during market hours)",
    interactive=False,
)

# Wire timer to refresh positions
price_timer.tick(fn=refresh_prices_callback, outputs=[positions_table])

# Manual trade entry (TRADE-03)
gr.Row:
    gr.Column(scale=1):
        gr.Markdown("### Manual Trade")
        ticker_input = gr.Textbox(label="Ticker", placeholder="e.g. AAPL", max_lines=1)
        shares_input = gr.Number(label="Shares", minimum=1, step=1, precision=0)
        gr.Row:
            buy_btn = gr.Button("Buy", variant="primary")
            sell_btn = gr.Button("Sell", variant="stop")
        trade_status = gr.Textbox(label="Trade Status", interactive=False)

# Trade history (TRADE-08)
history_table = gr.Dataframe(
    value=get_history_dataframe,
    headers=["Executed At", "Ticker", "Side", "Shares", "Price", "Realized P&L", "Notes"],
    label="Trade History",
    interactive=False,
)
```

Event wiring:
- set_capital_btn.click(fn=set_capital_callback, inputs=[starting_capital_input], outputs=[trade_status, cash_display, pnl_display, winrate_display])
- buy_btn.click(fn=buy_callback, inputs=[ticker_input, shares_input], outputs=[trade_status, positions_table, cash_display, pnl_display, winrate_display, history_table])
- sell_btn.click(fn=sell_callback, inputs=[ticker_input, shares_input], outputs=[trade_status, positions_table, cash_display, pnl_display, winrate_display, history_table])
