The leader ticker comes from the signal: for each position, look up the signal via source_signal_id (SELECT rowid AS signal_id, ticker_a FROM signals WHERE rowid = position.source_signal_id), then get the leader's latest abs(return_1d). Return positions where abs(return_1d) > invalidation_threshold, including ticker, shares, invalidation_threshold, leader_return_1d.

**1. Create paper_trading/engine.py:**

Constants at module top:
```python
SIZING_FRACTIONS = {'full': 0.20, 'half': 0.10, 'quarter': 0.05}
```

Functions (all take conn as first arg, use paper_trading.db helpers where possible):

- `set_capital(conn, starting_capital: float, portfolio_id: int = 1) -> dict`: (TRADE-01)
  Upserts paper_portfolio row with starting_capital and cash_balance = starting_capital. Uses `datetime.now(timezone.utc).isoformat()` for created_at/updated_at. Returns the portfolio dict. Uses INSERT with ON CONFLICT(portfolio_id) DO UPDATE SET starting_capital=excluded.starting_capital, cash_balance=excluded.cash_balance, updated_at=excluded.updated_at. This resets cash on re-set (per TRADE-01: "set starting paper capital").

- `compute_share_quantity(starting_capital: float, cash_balance: float, sizing_tier: str, entry_price: float) -> int`: (TRADE-02 helper)
  Exactly as in research. max_position_value = starting_capital * SIZING_FRACTIONS.get(sizing_tier, 0.10). affordable = min(max_position_value, cash_balance). Return int(affordable // entry_price) if entry_price > 0 else 0.

- `open_or_add_position(conn, portfolio_id, ticker, shares, price, source_signal_id, invalidation_threshold, executed_at) -> None`: (TRADE-02/03)
  Exactly as in research Pattern 3: record buy trade in paper_trades, upsert position with avg-cost formula via ON CONFLICT(portfolio_id, ticker) DO UPDATE, deduct cash from paper_portfolio. All in one transaction (conn.commit() at end). Notes field = 'auto_execute' if source_signal_id else 'manual'. CRITICAL: Cast shares to int before any DB write to avoid float precision issues.

- `close_position(conn, portfolio_id, ticker, shares_to_close, close_price, executed_at, notes='manual') -> float`: (TRADE-03/06)
  Exactly as in research Pattern 3: validate shares_to_close <= held_shares (raise ValueError if not). Compute realized_pnl = shares_to_close * (close_price - avg_cost). Record sell trade with realized_pnl. If remaining == 0 DELETE position, else UPDATE shares. Return cash. conn.commit(). Return realized_pnl.

- `auto_execute_signals(conn, api_key: str, portfolio_id: int = 1) -> list[dict]`: (TRADE-02)
  1. Get portfolio (raise ValueError if none). 2. Call get_unprocessed_signals(conn) from db.py. 3. For each signal: determine the follower ticker (the ticker that FOLLOWS -- based on direction from the signal's flow_map_entry; use ticker_b as the trade target). Fetch current price via fetch_snapshot_price(ticker_b, api_key). compute_share_quantity(). If shares == 0: skip (log insufficient cash). Call open_or_add_position(). Collect and return list of executed trade dicts. Wrap each signal execution in try/except to continue on individual failures.

- `get_portfolio_summary(conn, portfolio_id: int = 1) -> dict`: (TRADE-04/06)
  Exactly as in research code example: returns dict with cash_balance, starting_capital, unrealized_pnl (SUM of (current_price - avg_cost) * shares where current_price IS NOT NULL), realized_pnl (SUM from sell trades), win_rate (wins/total_closed * 100), total_pnl. Use COALESCE for NULL safety.

- `get_open_positions_display(conn, portfolio_id: int = 1) -> list[dict]`: (TRADE-04/07)
  Query paper_positions. For each position, compute unrealized_pnl = (current_price - avg_cost) * shares (or None if current_price is NULL). Call check_exit_flags from db.py to get flagged positions. Add 'exit_flag' boolean to each position dict. Return list of dicts suitable for DataFrame conversion.

- `get_trade_history_display(conn, portfolio_id: int = 1) -> list[dict]`: (TRADE-08)
  Thin wrapper around db.get_trade_history -- returns list of dicts from paper_trades ordered by executed_at DESC.

**2. Create paper_trading/price_poller.py:**

- Module-level: `_NYSE = None` (lazy-init singleton to avoid import-time expense)
- `_get_nyse()`: Lazy-initializes and returns the NYSE calendar singleton using pandas_market_calendars.get_calendar('NYSE').

- `is_market_open() -> bool`: (TRADE-05)
  Uses _get_nyse(). Gets now_et = pd.Timestamp.now(tz='America/New_York'). Gets schedule for today. Returns _NYSE.open_at_time(schedule, now_et). Wraps in try/except, returns False on any error (fail closed).

- `fetch_snapshot_price(ticker: str, api_key: str) -> float | None`: (TRADE-05)
  Exactly as in research Pattern 4. GET https://api.polygon.io/v2/snapshot/locale/us/markets/stocks/tickers/{ticker} with apiKey param. Fallback chain: lastTrade.p -> min.c -> day.c -> prevDay.c. Returns float or None. Uses requests.get with timeout=10. Logs errors via structlog.

- `poll_and_update_prices(conn, api_key: str, portfolio_id: int = 1) -> int`: (TRADE-05)
  1. Check is_market_open(). If not open, return 0. 2. Get all open position tickers from paper_positions. 3. For each ticker, call fetch_snapshot_price(). 4. Call update_position_prices(conn, prices_dict, now_utc). 5. Return count of updated positions. Log activity via structlog.
